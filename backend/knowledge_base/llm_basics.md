# 大语言模型基础

## 什么是大语言模型

大语言模型（Large Language Model, LLM）是基于深度学习技术，通过在海量文本数据上训练而成的神经网络模型。它们能够理解、生成和处理人类语言，在多种自然语言处理任务中表现出色。

## 发展历程

### 第一阶段：统计语言模型
- N-gram 模型
- 隐马尔可夫模型（HMM）

### 第二阶段：神经网络语言模型
- Word2Vec（2013）
- GloVe（2014）
- FastText（2016）

### 第三阶段：预训练语言模型
- ELMo（2018）
- BERT（2018）：双向编码器
- GPT 系列（2018-2023）：自回归生成模型

### 第四阶段：大语言模型时代
- GPT-3（175B 参数，2020）
- GPT-4（多模态，2023）
- LLaMA 系列（Meta，开源）
- Qwen 系列（阿里，开源）
- Claude 系列（Anthropic）

## 核心技术

### Transformer 架构
- 自注意力机制（Self-Attention）
- 多头注意力（Multi-Head Attention）
- 位置编码（Positional Encoding）
- 层归一化（Layer Normalization）

### 训练方法
- 预训练（Pre-training）：在大规模无标注文本上学习语言模式
- 监督微调（SFT）：使用标注数据调整模型行为
- 人类反馈强化学习（RLHF）：通过人类偏好优化输出质量
- 直接偏好优化（DPO）：更高效的偏好对齐方法

### 推理优化
- 量化（Quantization）：降低模型精度以减少内存占用
- KV Cache：缓存注意力计算结果加速生成
-  speculative decoding： speculative 解码加速
- 模型并行：将模型分布到多个 GPU

## LLM 的局限性

1. **知识截止**：训练数据有时间限制，无法获取最新信息
2. **幻觉问题**：可能生成看似合理但实际错误的信息
3. **推理能力**：复杂数学和逻辑推理仍有不足
4. **上下文窗口**：无法处理超长文本
5. **计算成本**：推理需要大量 GPU 资源

## RAG 如何解决 LLM 的局限

RAG（检索增强生成）通过引入外部知识库，有效解决了 LLM 的知识截止和幻觉问题：
- 实时检索最新信息
- 基于真实文档生成回答
- 提供信息来源可追溯
